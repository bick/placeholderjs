# Installation
PlaceholderJS is mainly supported for React applications. You can install it using npm or yarn.

npm:
<Code code="npm install placeholder" />

yarn:
<Code code="yarn add placeholder" />

pnpm:
<Code code="pnpm add placeholder" />
